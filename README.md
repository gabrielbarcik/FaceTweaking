# FaceTweaking


Application for tweaking a photo of a face. Ex: Making her smile, adding beard, changing eye color

## Authors

Cauim de Souza Lima
Gabriel Fedrigo Barcik
Victor Hugo Vianna

## Requirements

Java 1.8
OpenCV 3.2

## Instructions

Open a shell and run
```
$ sudo bash RUN.sh
```
